# Cleanup Patch 1 — Gameplay CSS Consolidation

## Active files
- `gameplay.variables.css`: shared variables.
- `gameplay.shared.css`: shared Gameplay components and utility rules.
- `gameplay.landscape.css`: desktop and mobile-landscape layouts.
- `gameplay.portrait.css`: all mobile portrait Gameplay rules.
- `gameplay.states.css`: turn, result, ZAI/timer and grayscale states.
- `gameplay.controls.css`: first place to edit positions and sizes.

The former Gameplay CSS files remain in the repository as inactive reference files, but `main.jsx` no longer imports them. Do not edit those inactive files.
